## 水平居中

### 案例 1

```css
.wrapper {
  width: 240px;
  height: 150px;
  background: #eee;
  margin-bottom: 20px;
}
.wrapper1 {
  text-align: center;
}
<div class="wrapper wrapper1">
  <span>行内，text-align: center;</span>
</div>
```

<div class="wrapper wrapper1">
  <span>行内，text-align: center;</span>
</div>

## 水平垂直居中

### 案例 1

```css
.wrapper {
  width: 240px;
  height: 150px;
  background: #eee;
  margin-bottom: 20px;
}
.wrapper9 {
  position: relative;
}
.wrapper9 div {
  width: 100px;
  height: 100px;
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  margin: auto;
}
<div class="wrapper wrapper9">
  <div>
    块级元素
    <p>已知宽高</p>
  </div>
</div>
```
