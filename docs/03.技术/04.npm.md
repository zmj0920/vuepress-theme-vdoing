### NPM 如何发布自己的升级包

npm unpublish --force

npm publish

### NPM 开启 cnpm 镜像

npm config set registry http://registry.cnpmjs.org //配置指向源,通过`nrm ls`查看源;

### npm 关掉 cnpm 镜像

npm config delete registry

npm config delete disturl

### NPM 查看当前用户

npm whoami

### 查看配置文件

npm config edit                             //打开.npmrc 文件;

### npm 安装指定版本

npm install lodash@3.8.0
